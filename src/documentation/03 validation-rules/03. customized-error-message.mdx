# Customized error message

You can add customized error messages for built-in rules.

Default error messages will be overridden  by your custom message.

Use the __CleverForm's__ static method __customErrorMessage__ API to override the default error messages.

>   CleverForm.customErrorMessage(messagesObject)

‚ö†Ô∏è You don't need to use the __new__ keyword when adding a custom error message.



## # Example
 customErrorMessage method accepts __[ruleName] : [customMessage]__ value pairs as shown below.

 ```js

// Customized error messages
CleverForm.customErrorMessage({

    required:'You must fill up :fieldName field.',
    minLen:'The :fieldName should be atleast :min characters.',
    maxLen: 'The :fieldName should not greater than :max characters.',
    // other rule's custom messages here that you want to override...

});


// CleverForm instance here....

 ```

 ‚ö†Ô∏è Customized error messages must be on top of CleverForm instances.

## # Placeholders
Error messages contain placeholders, they will be replaced depending on the validation details value.

Placeholders denoted by a colon (__:__) prefix.

One of the most important  placeholders is the __:fieldName__ that will be replaced by the field name value when the validation error message is emitted.
