# Options
__CleverForm__ _constructor function_ expects only one (1) paramater which is an __object__.

That object contains collection of __[optionName] / [value]__ pairs.

Below are the list of options.

| Name &nbsp;&nbsp; | Value type  &nbsp;| <center>Description</center>
| :---- | :-----:    | :-----:     |
| **id** |  _string_   | __Required__ - the id of HTML form that you want to have form validation |
| **rules** |  _object_   | __Required__ - Contains the fields validation rules |
| **customName** |  _object_   | __Optional__ -  holds every fields custom name in the error messages|
| **formEvents** |  _object_   | __Optional__ - holds every form events where you can listen to|
| **cssClass** |  _object_   | __Optional__ - hold CSS classes|
| **settings** |  _object_   | __Optional__ - hold form settings for its behavior.|
