## # NPM

NPM is the recommended installation method when building large scale applications.

[Visit CleverForm at NPM ](https://www.npmjs.com/package/cleverform)

```cli

$ npm install cleverform

```

__then...__

```js

// ES6
import CleverForm from 'cleverform'

// or...

// CommonJS
var CleverForm = require('cleverform')

```

## # Yarn
